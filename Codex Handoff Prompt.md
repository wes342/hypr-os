  ## Architecture                                                 
                                                                                                                                                                                                                                                            
  The system uses wallpaper-driven dynamic theming — colors are extracted from the current wallpaper via ImageMagick and applied to all apps. The key scripts:                                                                                              
   
  - **scripts/theme.sh** — Extracts 16 colors from wallpaper, generates configs for kitty, waybar, rofi, mako, eww, btop, cava, SDDM. Reloads all apps after.                                                                                               
  - **scripts/wallpaper.sh** — Picks random wallpaper (local/wallhaven/both), calls theme.sh. Supports --restore flag for boot persistence. Respects folder filter and wallhaven query from config.
  - **scripts/wallhaven.py** — Wallhaven API client for browsing/downloading wallpapers.                                                                                                                                                                    
                                                                  
  ## GTK4 Apps (Python + Adwaita)                                                                                                                                                                                                                           
                                                                  
  We've been migrating from rofi to GTK4 for interactive UIs:                                                                                                                                                                                               
                                                                  
  - **scripts/launcher-app.py** — App launcher (Super+Space). Transparent overlay with blurred wallpaper, Nerd Font sidebar icons (terminal/files/browser), searchable app list. Stays resident via D-Bus for instant reopen (~25ms). Pre-warm disabled by  
  default to save ~215MB RAM (toggle in settings app).
  - **scripts/wallpaper-app.py** — Wallpaper manager (Super+ALT+B). Three tabs: Local (folder filter dropdown), Wallhaven (sort buttons + search), Both (combined). Folder filters sync between tabs. Has auto-rotate timer via systemd, Wallhaven cache    
  cleanup, and theme toggle.                                                                                                                                                                                                                                
  - **scripts/settings-app.py** — Settings menu (Super+ALT+Space). Categories: Theme (terminal image picker — manual or follow wallpaper), Performance (launcher pre-warm toggle). Uses native GTK file dialog.

  ## Key Config Files                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                            
  - config/hypr/autostart.conf — Startup apps (waybar, hyprpaper, mako, eww, hypridle, wallpaper restore)                                                                                                                                                   
  - config/hypr/keybinds.conf — All keybindings                   
  - config/hypr/rules.conf — Window rules (floating, sensor panel protection, launcher)                                                                                                                                                                     
  - config/hypr/appearance.conf — Gaps, borders, animations, input, xwayland settings                                                                                                                                                                       
  - config/hypr/monitors.conf — DP-3 (2560x1440@180Hz primary), HDMI-A-2 (480x1920@60Hz sensor panel, transform 3)                                                                                                                                          
  - config/mako/config — Notification daemon (replaced swaync), auto-themed by theme.sh                                                                                                                                                                     
  - config/eww/ — Sensor panel widgets + waybar dropdown menus                                                                                                                                                                                              
  - config/kitty/kitty.conf — Terminal config with dynamic theme import                                                                                                                                                                                     
  - config/fastfetch/config.jsonc — System info with logo image (logo.png, 320x320, cached by fastfetch)                                                                                                                                                    
  - config/systemd/user/wallpaper-rotate.{service,timer} — Auto-rotate wallpaper

  ## Recent Work / Known Issues                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                            
  - Wallpaper browser Both tab filters can be flaky — tiles load in background threads, filter timing depends on 500ms timeout

  - Fastfetch caches images at ~/.cache/fastfetch/images/ — must clear after changing logo.png
